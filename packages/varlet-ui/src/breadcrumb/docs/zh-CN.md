## API

### 属性

| 参数        | 说明                                       | 类型                          | 默认值 |
| ----------- | ------------------------------------------ | ----------------------------- |-----|
| `separator` | 分隔符                                     | _string_                      | `-` |

### 事件

| 事件名  | 说明                                   | 参数                                          |
| ------- |--------------------------------------| --------------------------------------------- |
| `click` | 点击面包屑时触发 | `event: Event` |

### 插槽

| 参数        | 说明             | 参数 |
| ----------- | ---------------- | ---- |
| `default`   | 自定义默认内容   | `-`  |
| `separator` | 自定义分隔符内容 | `-`  |
