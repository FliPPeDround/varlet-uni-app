## API

### 属性

| 参数 | 说明 | 类型 | 默认值 |
| ----- | -------------- | -------- | ---------- |
| `name` | 唯一标识符，默认为索引值 | _number \| string_ | `index` |
| `title` | 面板标题 | _number \| string_ | - |
| `icon` | icon的名称 | _string_ | `chevron-down` |
| `disabled` | 是否禁用面板 | _boolean_ | `false` |

### 插槽

| 名称 | 说明 | 参数 |
| ----- | -------------- | -------- |
| `default` | 面板的内容 | - |
| `title` | 面板的标题 | - |
| `icon` | 自定义右侧icon | - |
